Credentials

Precog will need your credentials for accessing the API. Types of credentials differ from one API to the next. Check the API documentation to find out what type of credentials are required and what steps you will need to complete to attain those credentials. Sometimes credentials will be referred to as an ‘application’ or ‘app’. Often you will need to create an ‘application’ to connect you to your {Salesforce} data. This does not require you to develop or code an application but rather refers to creating the credentials required to access the API. We advise that you name any applications 'precog' so it will be clear what the credentials are being used for in the future.
